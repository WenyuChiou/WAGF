# Reference Data

This directory is reserved for external reference data that is still required by active experiments.

## Active Reference Data

- `CRSS_DB/`
  Required by the irrigation workflow and documentation for real-mode CRSS-based runs.
- `RL-ABM-CRSS/`
  Restored from the official Hung & Yang (2021) GitHub repository and required for
  the FQL baseline plus real-mode irrigation profile generation.

## Boundary Rule

- Keep only external reference assets that are still used by active runtime or analysis code.
- Historical prototype references should be removed from this directory or archived outside the main repo workspace.
