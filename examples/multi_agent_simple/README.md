# Multi-Agent Simple Tutorial

This directory is a teaching example, not a paper-grade benchmark.

It exists to show ABM developers how to build a small multi-agent WAGF
experiment with:

- two agent types
- phase ordering
- cross-type governance
- shared environment state
- mock LLM execution

Use this example when you want to learn the multi-agent API surface quickly.
For the full water-sector multi-agent reference implementation, use
`examples/multi_agent/flood/`.

Run:

```bash
python examples/multi_agent_simple/run.py
```
